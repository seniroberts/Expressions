Project Overview: This will depct a restaurant service with options to place an order, eat in, pick up and delivery.

Customers Capabilities
-----------------------
1. View available Menu Options 
2. Place an Order
3. Select dinning options:
    - Eat in
    - Pick up
    - Drop off (delivery)


Restaurant Capabilities
-----------------------

1. Receive Orders
2. Process Orders
3. Generate Customer Bill
